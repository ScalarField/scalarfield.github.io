A-Frame

I recently discovered A-Frame, a VR framework for web development, and I've been messing around in it. The link in this post's header takes you to the A-Frame site. A-Frame implements default WASD and click/drag camera controls for computer users, tilt controls for mobile users, and proper support for VR headsets/controllers. In this demo, just click the box with the on-screen cursor to increment the counter.
